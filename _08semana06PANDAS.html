Data frame¶

Para arrays de mayor dimensin (principalmente arrays bidimensionales, o tablas), la estructura de datos estructura de datos correspondiente es el objeto Pandas DataFrame. Se puede ver como una coleccin de objetos Series con un ndice comn.

Existen numerosas formas de inicializar un DataFrame. Para ejemplos sencillos, la forma ms fcil es pasar una lista anidada de Python o un diccionario al constructor del objeto DataFrame del objeto DataFrame. Por ejemplo, consideremos una extensin del conjunto de datos que utilizamos en la seccin anterior seccin anterior, en la que, adems de la poblacin de cada pas, incluimos una columna que especifica del idioma. Podemos crear el correspondiente objeto DataFrame de la siguiente manera:

Las filas de una instancia de DataFrame se puede acceder mediante el atributo loc indexer. La indexacin de este atributo tambin da como resultado un objeto Serie, que corresponde a una fila del DataFrame original:

In [12]:
dataFrame.loc['Mxico']
Out[12]:
Poblacin        126
Idioma       Espaol
Name: Mxico, dtype: object

Si se pasa una lista de etiquetas de filas al indexador loc, se obtiene un nuevo DataFrame que es un subconjunto del DataFrame original, que contiene slo las filas seleccionadas

In [15]:
dataFrame.loc[['Mxico', 'Colombia']]
Out[15]:
Poblacin Idioma
Mxico 126.0 Espaol
Colombia 51.0 Espaol

El indexador loc tambin puede utilizarse para seleccionar simultneamente filas y columnas pasando primero una etiqueta de fila (o una lista de ellas) y segundo una etiqueta de columna (o una lista de las mismas). El resultado es un DataFrame, una Serie, o un valor de elemento, dependiendo del nmero de columnas y filas que se seleccionen

In [17]:
dataFrame.loc[['Mxico', 'Colombia'],'Poblacin']
